# Sustainability Shell (React + Vite)

- 4 layout components (top header, left nav, right auth panel, footer)
- React Router routes render in the center (Home/About/Profile)
- Mobile: left/right become overlay drawers + backdrop tap-to-close
- Sidebars scroll independently when content is long

## Run
```bash
npm install
npm run dev
```

## Django auth
Replace endpoints/markup in:
